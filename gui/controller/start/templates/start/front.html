Solstice is an open source platform focused on the challenge of Sustainable Portfolio Management

Sustainability (both environmental and social) is a major issue that affects all economies. Developing tools and methodologies that improve the ability of all stakeholders to assess and report risks is an important objective and Solstice aims to help carry out this process as optimally as possible.

Solstice is a simulation framework

How does it work?

Running the application creates a server that can then be accessed via any regular web browser. NB: For easy installation and testing the current release uses a self-contained sqlite database instead of a production database server

The initial functionality of Solstice focuses on integrating the following concepts

  • Credit Networks
  • Input-Output Netoworks

Once the platform is up and running a wealth of functionality is available.

User Functionality

  • Inspect available workflows
  • Execute workflows
  • Inspect and Export Simulation results
  • Create visualizations

Administrator Functionality

  • An administrator can create workflows
  • Perform bulk insertion or deletion of data sets

Developer Functionality

  • Solstice is a C++ application with a pluggable architecture
  • Developers can customize and adapt the platform in unlimited ways
